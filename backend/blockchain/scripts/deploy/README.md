# Contract Deployment Scripts

Scripts for deploying smart contracts to the Besu network.

## Scripts

- `deploy-contract.js` - Deploy CertificateManager contract to Besu

## Usage

### Deploy Contract

```bash
# Install dependencies
npm install web3

# Deploy contract
node blockchain/scripts/deploy/deploy-contract.js
```

## Note

For production deployment, use:
- Remix IDE: https://remix.ethereum.org
- Hardhat or Truffle
- Or compile and deploy via go-ethereum bindings

The `deploy-contract.js` script is a template showing the deployment process.
